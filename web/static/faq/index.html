Seekly FAQ

How does Seekly work?

Seekly is a meta search engine, meaning it scrapes data from other search engines to get it's results.
Seekly scrapes specifically from Bing, the second web largest search engine in the world.
Seekly also protects your privacy by not letting your browser tell the website where you came from.

How much of the source code is available to the public?

100% of it is. We have opted to keep 100% of the website's code (frontend and backend) open-sourced and free
for anybody to use, see the license we use for more information.

What privacy extensions/add-ons would you recommend?

Glad you asked, here's a few I, normanlol on github, recommend:

  1. uBlock Origin (adblocker that also blocks trackers) [Firefox] [Chrome]
  2. Privacy Badger (EFF-made tracker blocker) [Firefox] [Chrome]
  3. Decentraleyes (other tracker blocker) [Firefox] [Chrome]
  4. Smart Referer (doesn't tell new sites where you came from) [Firefox]
  5. Universal Bypass (securely bypasses link shorteners that track you) [Firefox] [Chrome]
We are not paid by any developers to list or endorse any of the extensions above.

Is this project taking donations/contributions?

As of now, the Seekly project does not take direct donations. We may at some point.

If you want to contribute, if you have a general understanding of NodeJS and scraping with Cheerio, please feel free to contribute
to either the main project or the scraper behind it.
